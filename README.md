# 命名规范


1. 文件命名

- C++头文件: `MyFile.hpp`
- C++源文件: `MyFile.cpp`

1. 类型(class/struct/enum/typedef/using)

- `class MyEngine;`

3. 变量

- 普通变量: `int table_name`
- 类成员变量: `m_table_name`
- 结构体成员变量: `table_name`

4. 常量

- `const kMaxPoolSize = 10;`

5. 函数命名

- `void MyFunction();`

6. 枚举值

```
enum MyEnum {
	None = 0,
	Windows,
	Linux
};
```

7. 宏

- `#define MY_DEBUG_FLAG`
